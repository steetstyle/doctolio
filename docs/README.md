# Kendimi nasıl tanımlarım

> İstanbul'da yaşayan öğrenmeye bağımlı bir yazılım geliştiricisiyim. Yazılım teknolojilerinin gelişiminin bir parçası olmayı seviyorum. Ayrıca fizik öğrencisiyim, burada kendi öğrendiğim ve öğrenirken karşılaştığım zorlukları daha kolay ve anlaşılır hale getirip yazıp dahada iyi anlamaya çalışacağım...
